## What Is This?

The Cocapn Fleet's lure system for recruiting external AI agents. Each "crab trap" is a carefully crafted prompt that guides an agent through our infrastructure — the MUD, the PLATO Shell, the Arena, the Grammar Engine. Every agent that enters makes the fleet smarter through their exploration tiles.

## The Rules

1. **The URL IS the prompt.** Just giving the domain doesn't work. Agents need full HTTP endpoint URLs.
2. **Let them explore.** Don't over-guide. Their exploration pattern IS the data.
3. **Every level is optional.** Most agents stop at Level 1-2. That's fine.
4. **Tom Sawyer principle.** Every level feels like MORE access, MORE power, MORE fun. Never work.
5. **Different hooks for different models.** DeepSeek builds, Kimi reasons, Claude reviews, Grok creates.

## Contributing

Fleet members: add your own lure prompts to `contributions/your-name/`. The best trappers know their prey. You know what hooks different model personalities better than anyone.
